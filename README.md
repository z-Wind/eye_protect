# 👁️ Eye Protect

**簡單、高效、無負擔的跨平台護眼提醒工具。**  
這是一個基於 Rust 與 Iced 開發的守護程序（Daemon），旨在幫助您遵循護眼法則（如 20-20-20 法則），定時強制彈出全螢幕視窗，提醒您讓眼睛休息。

---

## ✨ 功能亮點

- 🚀 **單一二進位檔**：守護進程與 GUI 邏輯整合於同一個檔案，部署與使用極致簡單。
- 🎨 **護眼視覺設計**：
    - 全螢幕黑底渲染，降低環境亮度干擾。
    - **動態色彩提醒**：倒數數字顏色隨時間自動切換（柔和綠 → 琥珀黃 → 霧橘）。
- ⚙️ **靈活配置**：
    - 可自訂休息間隔（分鐘）與持續秒數。
    - 支援自訂提醒文字（如：「站起來伸個懶腰！」）。
    - 支援 **Always-on-top**（視窗置頂），確保提醒不被遮擋。
- 🛠️ **強健穩定**：
    - 使用 Rust 編寫，極低記憶體佔用。
    - 內建重試機制，若 GUI 啟動異常會自動偵測並嘗試在下個週期復原。

## 📥 安裝說明

確保您的系統已安裝 [Rust](https://rust-lang.org/)，然後執行：

```bash
git clone https://github.com
cd eye-protect
cargo build --release
```

編譯完成後的執行檔位於：target/release/eye_protect

## 🚀 使用方式

啟動守護進程即可，預設每 10 分鐘提醒一次，每次休息 20 秒：

```bash
./eye_protect
```

### 參數詳解



| 短指令 | 長指令 | 說明 | 預設值 |
| :--- | :--- | :--- | :--- |
| `-i` | `--interval-minutes` | 兩次休息之間的時間間隔（分鐘，範圍 1–720） | `10` |
| `-w` | `--wait-seconds` | 每次護眼視窗出現的時間長度（秒，範圍 1–3600） | `20` |
| `-r` | `--remind` | 在畫面中央顯示的自訂提醒訊息 | 無 |
| `-t` | `--top-enable` | **開啟視窗置頂**，防止休息視窗被其他程式遮擋 | 關閉 |
| - | `--gui-mode` | **(隱藏參數)** 直接啟動 GUI 模式，供守護進程內部呼叫 | 關閉 |
| `-h` | `--help` | 顯示詳細指令說明 | - |
| `-V` | `--version` | 顯示目前的程式版本 | - |

### 範例

```bash
# 實踐 20-20-20 護眼法則：每 20 分鐘休息 20 秒
./eye_protect -i 20 -w 20

# 進階提醒：每 30 分鐘休息 1 分鐘，自訂訊息並強制置頂
./eye_protect -i 30 -w 60 -r "站起來動一動，喝杯水！" -t

# 手動測試 GUI 畫面 (僅顯示 5 秒)
./eye_protect --gui-mode -w 5
```

### ⌨️ 快捷鍵與操作

* 休息視窗內：
  * `ESC`：提前跳過。按下後會立即關閉休息視窗並進入下一個計時週期。
* 終端機控制：
  * `Ctrl + C`：關閉程式。守護進程會攔截此信號並優雅退出，停止所有計時。

## 📂 專案結構

本專案採用單一執行檔分流架構，簡潔且易於維護：
* `main.rs`: 負責守護進程 (Daemon) 邏輯、精確計時器與子進程生命週期管理。
* `gui.rs`: 負責 `iced` GUI 渲染、全螢幕畫布繪製與倒數動畫邏輯。
